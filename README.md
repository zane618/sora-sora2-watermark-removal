# sora-sora2-watermark-removal [项目地址](https://github.com/zane618/sora-sora2-watermark-removal)
sora2视频秒去水印

sora2很强大，但是有水印很不方便，我找到了一个去sora2水印的微信小程序{ 一键宝 }，分享给大家
1. 把sora发布的视频链接放进去，点击【开始去除水印】
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/9d5ff81fa736472aa7b39441101f0328.png#pic_left)
2. 点击【复制链接】，去浏览器下载即可
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/bd3266cac8524323b4614693ba4b9616.png#pic_left)
